Help save iGoogle by putting a "Save iGoogle" ribbon on your WordPress site or network that links to the Save iGoogle campaign.

== Description ==

A black ribbon with the words "Save iGoogle" will be put in the top right corner of your website linking to the [Save iGoogle Campaign webpage](http://saveigoogle.org/save-igoogle-campaign/) at SaveiGoogle.org. Look at the screenshot to see how the ribbon looks. Check out the [SaveiGoogle.org blog](http://saveigoogle.org/blog/) too. Thank you for showing your support!

== Installation ==

1. Download the plugin zip archive.
2. Extract it in your `wp-content/plugins` folder or upload via admin panel.
3. Browse to your plugins section and activate the plugin.
